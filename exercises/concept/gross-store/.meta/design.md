# Design

## Goal

The goal is to introduce the student to `Hash` as the class, and as a concept.

## Learning objectives

- initializing `Hash` instances
- setting key-value pairs
- modifying values for existing keys
- deleting key-value pairs
- reading a key, non-existent key returns zero value of value type

## Out of scope

- iterate over Hash

## Concepts

- Hashes
## Prerequisites

- enumerations
- arrays
- strings
- conditionals
- numbers
