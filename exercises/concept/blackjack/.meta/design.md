# Design

## Goal

The goal of this exercise is to teach the student about cases in Ruby.

## Learning objectives

- Know about subsumption operator
- Know about case expressions
- Know about single line case expressions

## Out of scope

- Regex with cases

## Concepts

`case`:

- Case subsumption operator (`===`)
- `case`.
- `when` and `then`.
- Implicit `when`.

## Prerequisites

- Modules
- Ranges
